# Conway-s-Game-of-Life---VBA
A Macro Script that simulates Conway's Game of Life within an Excel Worksheet.

## Features

- Configurable board size
- Configurable random seed
- Adjustable initial density
- Start/Pause/Step controls
- Manual editing by double-clicking
- Save/load custom patterns through .txt uploading and inserting
- Text-based pattern format

## Skills Demonstrated

- VBA programming
- Excel automation
- Dynamic array allocation
- Event-driven programming
- File I/O
- Serialization/deserialization
- Algorithm implementation

## How to Run

1. Enable Macros (Right Click the downloaded file, properties, check unblock, apply. You may also have to enable macros on the yellow banner when you first open the file.).
2. Open GameOfLife.xlsm.
3. Configure settings. If you do not want any seed, set seed to -1. This will generate a random board each time.
4. Click Seed.
5. Click Begin.
6. To edit the board at any time, pause the game and double click on a cell to flip its status.
